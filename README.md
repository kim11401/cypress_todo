## 프로젝트 목적
Cypress를 사용하여 E2E 테스트 코드와 CI/CD를 학습 및 실습해보는 프로젝트 입니다.
프로젝트 실습을 위해 React를 사용하여 간단한 TodoList를 구현하였습니다.
